# TRX Launcher

A sleek cross-platform launcher for the TRX engine, supporting **TR1, TR1 Unfinished Business, TR2, TR2 Golden Mask, TR3, TR3 Lost Artifact**. Built with **Electron**, HTML, CSS, and JS.  

It features:  

- **Dynamic screen navigation** (TR1/TR2/TR3) with smooth keyboard arrow transitions  
- **Crossfade background music** with mute toggle  
- **Last played game auto-detection**  
- **TRX executable management** (auto-detect or manual selection)  
- **Minimal UI**: no toolbar, no scrollbars  
- **Cross-platform support**:  
  - Windows `.exe`  
  - Linux executable  
  - macOS `.app`  

---

## 🎮 Requirements

- Node.js >= 18.x  
- npm (or yarn)  
- TRX engine compiled and ready  
- Electron installed with npm
